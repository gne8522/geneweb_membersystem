2. 基礎的會員資料CRUD<br>
● 註冊表單時，填入的帳號，會跟資料庫做檢查，重複則不能使用 <br>
● 同個登入頁面，輸入工作人員帳號則進入後台管理系統<br>
● 客戶更改大頭照時有簡易預覽畫面<br>
● 後台可對客戶的資料作更動，前台則有限制，像是客戶不能自行調整會員等級跟抽獎次數，這些只能在後台能操作
